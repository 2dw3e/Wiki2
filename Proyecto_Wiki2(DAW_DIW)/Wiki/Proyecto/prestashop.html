Instalación

- Para instalar Prestashop lo primero que tenemos que hacer es entrar al servidor y ejecutar los siguientes comandos:

  • mysql -u root -p
  • mysql > CREATE DATABASE prestashop;
  • mysql > GRANT ALL ON prestashop.* TO 'root'@'localhost' IDENTIFIED BY 'Admin123_';
  • mysql > FLUSH PRIVILEGES;
  • mysql > EXIT;
    •
  • sudo apt-get update
  • sudo apt-get install php5 libapache2-mod-php5 php5-mysql php5-curl php5-gd php5-intl php-pear php5-imagick php5-imap php5-mcrypt php5-memcache php5-ming php5-ps php5-pspell php5-recode php5-snmp php5-sqlite php5-tidy php5-xmlrpc php5-xsl -y
  • sudo systemctl restart apache2
  • cd /var/www/html
  • wget prestashop.com/download/prestashop 1.6.1.5.zip
  • sudo apt-get install unzip -y
  • sudo unzip prestashop_1.6.1.5.zip
  • sudo chown -R www-data:www-data.*
  • nano /var/www/html/wp-config.php

    • define('DB_NAME', 'prestashop');
    • /** MySQL database username */
    • define('DB_USER', 'root');
    • /** MySQL database password */
    • define('DB_PASSWORD', 'Admin123_');

- Ahora vamos a instalar Prestashop a través del navegador web:

  • Entramos en “http://10.9.53.109/prestashop”
  • Buscamos el “Español (Spanish)”. Y le damos a “Siguiente”
  • Aceptamos los términos y condiciones arriba indicados y le damos a “Siguiente”
  • Rellenamos con los datos necesarios y le damos a “Siguiente”
  • Rellenamos con los datos de la base de datos que hemos creado antes y le damos a “Siguiente”
  • Esperamos a que se realice la instalación
  • Si intentamos acceder nos dará error, asi que tendremos que volver al servidor y borrar la carpeta de "install" y renombrar la carpeta de "admin"

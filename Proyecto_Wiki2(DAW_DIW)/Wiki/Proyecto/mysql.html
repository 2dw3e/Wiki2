Instalación

- Para instalar MySQL lo primero que tenemos que hacer es entrar al servidor y ejecutar los siguientes comandos:

  • sudo apt-get upgrade
  • sudo apt-get update
  • sudo apt-get install mysql-server


  • Entramos a localhost/phpmyadmin
  • http://localhost/phpmyadmin/index.php?token=0ee7b49a2ddaa5f73e8269eac93c9958
  • - Usuario → root
  • - Contraseña → admin

  • GITHUB – GIT (Comandos)

  • Dentro de la carpeta donde esté el proyecto, para subir los cambios:
  • git remote add origin https://github.com/2dw3e/Wiki2.git
  • git init
  • git add .
  • git commit -m "nombrecommit"
  • git push -u origin master

Instalación

- Para instalar Wordpress lo primero que tenemos que hacer es entrar al servidor y ejecutar los siguientes comandos:

  • mysql -u root -p
  • mysql > CREATE DATABASE wordpress;
  • mysql > GRANT ALL ON wordpress.* TO 'root'@'localhost' IDENTIFIED BY 'Admin123_';
  • mysql > FLUSH PRIVILEGES;
  • mysql > EXIT;
    •
  • sudo apt-get update
  • sudo apt-get install php-curl php-gd php-mbstring php-mcrypt php-xml php-xmlrpc
  • sudo systemctl restart apache2
  • sudo nano /etc/apache2/apache2.conf
    • <Directory /var/www/html>
    • AllowOverride All
    • </Directory>
  • sudo a2enmod rewrite
  • sudo systemctl restart apache2
  • cd /var/www/html
  • wget https://wordpress.org/latest.tar.gz
  • tar xzvf latest.tar.gz
  • sudo chown -R www-data:www-data.*
  • nano /var/www/html/wp-config.php

    • define('DB_NAME', 'wordpress');
    • /** MySQL database username */
    • define('DB_USER', 'root');
    • /** MySQL database password */
    • define('DB_PASSWORD', 'Admin123_');

- Ahora vamos a instalar Wordpress a través del navegador web:

  • Entramos en “http://10.9.53.109/wordpress”
  • Le damos a “¡Vamos a ello!”
  • Rellenamos con los datos necesarios y le damos a “Enviar”
  • Ahora ponemos el título, el nombre de usuario de WordPress, una dirección de correo y una contraseña, y le damos a “Instalar WordPress”
  • Y listo ya tenemos WordPress instalado. Le damos a “Acceder”
  • Nos logueamos con el usuario y la contraseña que hemos puesto antes.
  • Nos aparecerá el panel de control desde el que podremos editar y personalizar a nuestro gusto el WordPress.
